== Description ==

This plugin simply Insert post's or page's title in content area. If you are really sick of copying and pasting title in content again and again, then simply use this plugin because it comes with Shortcode button. Click on button from editor to insert the title of the page.

= Features =

* Insert **"Post's or Page's"** title easily and use it within Content area
* Insert Shortcode using Visual Editor or Text Editor
* Simple Shortcode functionality
* SEO Friendly
* Mobile and Mobile theme's friendly
* Simple lightweight Plugin
* Easy to use

== Installation ==

If you are on WordPress website then: 

1. Go to **Plugins** and click on **Add new**.
1. Search for Plugin **"Insert Title by Harman Singh Hira"** in the directory.
1. Now, **Install and Activate** it.

Or, if you want to download it in your machine and want to upload it via FTP then do according to following below steps:

1. Download Plugin in your machine
1. Extract the zipped file and upload the folder `ut-insert-title` to `/wp-content/plugins/` directory.
1. Activate the plugin through the `Plugins` menu in WordPress.
